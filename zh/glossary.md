- DNR - 数字降噪
- DNS - 域名服务

- DVR - 数字视频录像机
- FTP - 文件传输协议
- HDR - 高动态范围
- HLS - HTTP 实时流媒体（协议）

- NVR - 网络视频录像机

- HTTP - 超文本传输​​协议

### ffmpeg/ffplay

- PTS - 演示时间戳
- fps - 每秒帧数的平均帧速率（`AVStream.avg_frame_rate`）
- tbr - 流的实际基准帧速率（`AVStream.r_frame_rate`）
- tbn - 每秒刻数的时间刻度（`AVStream.time_base`）
