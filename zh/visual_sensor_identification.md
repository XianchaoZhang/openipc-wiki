# 传感器特写 
-----------------

可用于直观地识别实际传感器，例如验证在线卖家是否确实向您出售广告中的传感器。有时，查看比在新购买的 ipc 上运行 ipctool 更快。

令人感兴趣的是键合线和焊盘的位置。
